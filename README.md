# UltiSID
Open source SID 8580 emulator that can be executed on Bluepill boards

Roadmap: 
transfer to GCC embedded workbench and optimize performance, fix interrupt errors. 

Actual branch: hal_usid

Based on STM32-SID-PLAYER
